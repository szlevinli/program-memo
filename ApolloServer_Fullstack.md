# ApolloServer Fullstack

## 需要的包 (packages)

### 用于产品

> - **apollo-datasource:** 官方提供的数据源基础包
> - **apollo-datasource-rest:** 社区提供的基于 RestFul 数据源的包
> - **apollo-server: **官方包
> - **apollo-server-testing: **测试包
> - **dotenv: **loads environment variables from a `.env` file into `process.env`
> - **graphql: **GraphQL
> - **isemail:** Node email address validation library
> - **nodemon: ** runs TypeScript compiler in watch mode, and restarts Node.js when TypeScript files are modified.
> - **sequelize:** Sequelize is a promise-based Node.js ORM for Postgres, MySQL, MariaDB, SQLite and Microsoft SQL Server.
> - **sqlite3:** for SQLite

### 用于开发

> - **apollo-link:** is the interface for creating new links in your application.
> - **apollo-link-http:** Get GraphQL results over a network using HTTP fetch.
> - **jest:** 测试框架
> - **nock:** HTTP server mocking and expectations library for Node.js
> - **node-fetch:** A light-weight module that bring `window.fetch` to Node.js
